## QUICK REFERENCE

### If a page isn't indexing:
1. Check Google Search Console → Coverage → See why
2. Request indexing again
3. Check robots.txt allows page (it does ✅)
4. Wait 2-4 weeks (Google indexes on its schedule)

### If rankings aren't improving:
1. Keep adding fresh blog content (1-2 per week)
2. Build backlinks (guest posts, mentions)
3. Improve content depth & quality
4. Monitor Core Web Vitals (keep them green)

### If traffic drops:
1. Check for crawl errors in GSC
2. Verify pages still exist (no 404s)
3. Check robots.txt settings
4. Contact Vercel support if site is down
